# Final_SS2
Project Description

Budget Tracking App for Students is a web application designed to help university students manage their personal finances more effectively. The system allows users to record their income and expenses, organize transactions by category, set monthly spending limits, and monitor their financial situation through charts and summary reports.

The main goal of this project is to improve students’ financial awareness and support better money management habits in daily life. Each user has a personal account and a private dashboard where they can track their own financial data securely.

The application includes essential features such as Google OAuth2 login, transaction management, category management, monthly budget tracking, and data visualization. Users can also search and filter transactions based on date, category, type, or keywords. In addition, the system supports exporting transaction history as a CSV file for reporting purposes.

This project is built with a modern web development stack, using React for the frontend, Node.js with Express for the backend, and a relational database such as SQLite, MySQL, or PostgreSQL for data storage.
